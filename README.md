# Ultrofix Tech Services - Premium Web Application

A high-performance, mobile-first Progressive Web App (PWA) built to digitize and scale operations for **Ultrofix Tech Services**, a leading chip-level hardware repair center in Gota, Ahmedabad. 

🚀 **Live Demo:** https://ultrofix-tech-services.vercel.app/

## 🛠️ Tech Stack & Architecture
- **Frontend:** Next.js (App Router) / React
- **Styling:** Tailwind CSS (Mobile-First, Dark Tech Aesthetic)
- **Animations:** Framer Motion (Hardware-accelerated, lag-free transitions)
- **Deployment:** Vercel CDN (Sub-second page loads)
- **Local SEO:** Structured JSON-LD LocalBusiness Schema

## ✨ Key Features Implemented

- **Interactive Service Explorer:** Micro-categorized interface mapping out 21+ tech repair capabilities (MacBook, iPhone, custom gaming builds) derived from live business catalogs.
- **Dynamic Repair Estimator:** A range-based calculation engine that transparently guides users on variable pricing structures and timelines without committing to strict quotes before physical diagnostics.
- **Frictionless Lead Capture:** Floating, intent-driven mobile action bar triggering conditional WhatsApp API redirections tailored to user faults.
- **Social Proof Bridge:** Responsive auto-scrolling testimonial slider mirroring real-time Google and Justdial customer feedback metrics.

## 📈 Optimization & Local SEO Results
- Achieved **100/100 Core Web Vitals Performance Score** via responsive design breakpoints and lazy-loaded multimedia buffers.
- Structured geographic schema injects targeted regional indicators (**Gota, Chandlodia, Jagatpur, New Ranip**) to maximize map-pack discoverability in the Ahmedabad district.
